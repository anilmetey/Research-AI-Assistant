# Research-AI-Assistant

Tamamen yerel veya bulut destekli çalışan, belgelerinizi yükleyip analiz ettirebileceğiniz, görsel, ses ve video oluşturma yeteneklerine sahip gelişmiş bir yapay zeka araştırma asistanı.

## Kurulum
1. `backend` klasöründe `npm install` ve `npm run dev`
2. `frontend` klasöründe `npm install` ve `npm run dev`
