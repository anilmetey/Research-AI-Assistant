<h2 id="english">🇬🇧 English</h2>

**Research AI Assistant** is a modern, full-stack AI platform designed to accelerate your research workflow. It allows you to upload documents, analyze them using advanced AI (RAG), generate stunning images, and create professional voiceovers and cinematic videos using state-of-the-art generative models.

### ✨ Key Features
- **Intelligent Chat & RAG:** Deep document analysis with conversational AI.
- **Ultra-Fast Cloud AI:** Lightning-fast text generation powered by cloud infrastructure.
- **Creative Tools Studio:**
  - **Image Generation:** High-quality, fast image generation using Pollinations AI.
  - **Video Generation:** Cinematic video creation using Luma Dream Machine API.
  - **Audio & TTS:** Natural text-to-speech capabilities.
- **Modern UI/UX:** Sleek, responsive, and animated interface built with Next.js and Framer Motion.

### 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), React, Framer Motion, Lucide Icons.
- **Backend:** Node.js, Express, Prisma (PostgreSQL), Redis.
- **AI Models:** Pollinations AI, Luma AI, local embedding integrations.

### 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anilmetey/Research-AI-Assistant.git
   cd Research-AI-Assistant
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   # Set up your .env file with DATABASE_URL, REDIS_URL, etc.
   npm run dev
   ```

3. **Frontend Setup:**
   ```bash
   cd frontend
   npm install
   # Set up your .env.local file (e.g., LUMA_API_KEY)
   npm run dev
   ```

4. Open `http://localhost:3000` in your browser.

<h2 id="turkish">🇹🇷 Türkçe</h2>

**Research AI Assistant**, araştırma süreçlerinizi hızlandırmak için tasarlanmış modern bir tam yığın (full-stack) yapay zeka platformudur. Belgelerinizi yüklemenize, onları gelişmiş yapay zeka (RAG) ile analiz etmenize ve muazzam görseller, profesyonel seslendirmeler ile sinematik videolar üretmenize olanak tanır.

### ✨ Öne Çıkan Özellikler
- **Akıllı Sohbet & Belge Analizi (RAG):** Belgelerinizle konuşun ve derinlemesine analiz yapın.
- **Ultra Hızlı Bulut Yapay Zeka:** Bulut altyapısıyla desteklenen şimşek hızında metin üretimi.
- **Yaratıcı Araçlar Stüdyosu:**
  - **Görsel Üretimi:** Pollinations AI ile yüksek kaliteli, anında görsel oluşturma.
  - **Video Üretimi:** Luma Dream Machine API kullanarak sinematik video tasarımı.
  - **Seslendirme (TTS):** Doğal dilde metin okuma ve seslendirme yetenekleri.
- **Modern Arayüz (UI/UX):** Next.js ve Framer Motion ile inşa edilmiş, karanlık mod destekli akıcı ve şık arayüz.

### 🛠️ Teknolojiler
- **Önyüz (Frontend):** Next.js (App Router), React, Framer Motion, Lucide Icons.
- **Arka Plan (Backend):** Node.js, Express, Prisma (PostgreSQL), Redis.
- **Yapay Zeka:** Pollinations AI, Luma AI ve yerel vektör entegrasyonları.

### 🚀 Kurulum

1. **Projeyi indirin:**
   ```bash
   git clone https://github.com/anilmetey/Research-AI-Assistant.git
   cd Research-AI-Assistant
   ```

2. **Arka Plan (Backend) Kurulumu:**
   ```bash
   cd backend
   npm install
   # .env dosyanızı DATABASE_URL, REDIS_URL vb. ile yapılandırın
   npm run dev
   ```

3. **Önyüz (Frontend) Kurulumu:**
   ```bash
   cd frontend
   npm install
   # .env.local dosyanızı yapılandırın (örn: LUMA_API_KEY)
   npm run dev
   ```

4. Tarayıcınızda `http://localhost:3000` adresini açın.
